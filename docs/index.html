What the demo shows

These screenshots capture the full local Cloud Shell run: the Spring Boot service starts, the user runs the advisor pipeline, Gemini searches and scores model candidates, and the app returns a practical recommendation for a RAG chatbot use case.

  1. Orchestrator service starts and logs each agent handoff.
  2. User run parses requirements and discovers model candidates.
  3. Recommendation output explains the primary pick and trade-offs.
  4. Build completes successfully after the pipeline run.

Pipeline flow
